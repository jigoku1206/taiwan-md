## 🎯 專案願景 | Project Vision

Taiwan.md 是一個**開源、AI-friendly** 的台灣知識庫，致力於：

- 📚 **完整性**：涵蓋台灣的歷史、地理、文化、美食等各個面向
- 🎭 **策展式內容**：不是百科全書，而是精心策展的深度內容
- 🌍 **國際視野**：中英雙語，用國際語言說台灣故事
- 🤖 **AI-Friendly**：結構化 Markdown，讓人工智慧也能理解台灣
- 🔓 **開源共享**：CC BY-SA 4.0 授權，促進知識自由流動

---

## 🔍 為什麼需要 Taiwan.md？

### 現有資源的不足

1. **碎片化資訊**：台灣相關資訊散落各處，缺乏系統性整理
2. **觀光導向**：多數資源以觀光為主，缺乏深度文化內容
3. **語言障礙**：優質中文內容缺乏英文對照，限制國際傳播
4. **AI 理解困難**：現有內容格式不利於 AI 處理和理解

### Taiwan.md 的解決方案

- ✅ **系統化組織**：12+ 主要面向的完整知識架構
- ✅ **三層深度設計**：30秒概覽 → 5分鐘理解 → 完整深度資料
- ✅ **雙語並進**：中英文同步更新，促進跨文化理解
- ✅ **結構化內容**：標準化 Markdown 格式，AI-friendly

## 🚀 快速開始 | Quick Start

### 環境需求

- Node.js 18+
- npm 或 bun

### 安裝與運行

```bash
# 複製專案
git clone https://github.com/frank890417/taiwan-md.git
cd taiwan-md

# 安裝依賴
npm install
# 或使用 bun
bun install

# 開發模式
npm run dev
# 或
bun run dev

# 建置專案
npm run build
# 或
bun run build
```

瀏覽器打開 `http://localhost:4321` 即可看到網站！
